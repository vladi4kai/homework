PHP та JavaScript

Мови програмування PHP та JavaScript дозволяють зробити сайт динамічним, тобто таким, що реагує на дії користувача. Наприклад, можна зробити гарне меню, що випадає, або слайдер

Види скриптів

Для цього пишуться скрипти (англ. script - "сценарій") - програми, що дозволяють реагувати на дії користувача. Скрипти бувають двох видів:

  • ті, які виконуються на сервері, а результат їхнього виконання приходить у браузер до користувача вже в готовому вигляді. Це скрипти, написані мовою PHP. На ньому пишуться CMS-ки - системи управління контентом
  • ті, які виконуються прямо у браузері користувача. Це скрипти, написані мовою JavaScript. Вони найчастіше використовуються для того, щоб зробити сторінку більш зручною та красивою
